# Hildegard — Source Journals

Hildegard surveils PubMed for recently published, practice-relevant papers (phase II/III trials and guidelines). Every search is **restricted to the journals listed below** — nothing outside this set is retrieved. Journal names, PubMed filter terms, and ISSNs were each verified live against PubMed.

**How the tiers are used**

- **Tier 1 — core clinical journals** are searched for *every* disease group, hematologic and solid-tumor alike.
- **Tier 2 — Hematology** is added for the hematologic disease groups.
- **Tier 2 — Oncology** is added for the solid-tumor disease groups (plus *Journal of Hematology & Oncology*, which spans both).

In the newest-paper "fresh scan," Tier 1 and Tier 2 are searched separately and written to separate files, so the higher-signal Tier 1 results stay clean.
